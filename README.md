本專案為利用2.9"電子紙搭配ESPhome與Home Assistant，平常顯示為時鐘與溫溼度

透過HA控制換頁，加上任何想要顯示的資訊，例如當衣服洗好時，會自動將畫面切為訊息提示

要有多少頁或是想要顯示什麼都能自由設計
